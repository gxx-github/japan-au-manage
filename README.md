



## 项目介绍

- 使用技术 React.js
- 使用脚手架 [UmiJS 4.x](https://umijs.org/docs/introduce/introduce)

## 分支说明
- **main** umi+less+rainbow+wagmi+px
- **dev** 


## 环境依赖

- Node 14+

## 安装项目依赖

```
npm install
```

## 项目启动

```
npm run dev
```

## 项目打包
测试环境打包，主要为了兼容测试链
```
npm run build:test
```
生产环境打包，主要为了兼容主网链
```
npm run build
```
